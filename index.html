README.md

Hi, I'm fightheyyy.

我正在围绕 XiaoBa-CLI 这个 Agent Runtime，尝试构建一套更开放的 Agent infrastructure。 它不只是一个命令行工具，而是一个可以承载多角色协作、工具调用、skills 扩展、长期记忆和多端接入的基础层。

我希望围绕这个 Runtime 打造一个 Agent 宇宙：不同 Agent 可以拥有独立上下文和执行生命周期， 可以通过 subagent 与 Agent2Agent 机制完成复杂任务拆解，也可以接入飞书、微信、桌面端、 Web 服务甚至硬件设备，成为更自然的数字伙伴。

接下来我会继续探索 XiaoBa-CLI 与硬件结合的可能性，比如开源桌宠、实体陪伴设备、 本地自动化控制和多模态交互，让 Agent 不只停留在聊天窗口里，而是能真正进入日常工作流和物理空间。

  • 开源 Agent infra：runtime、session、tools、skills、roles、permissions。
  • Multi-agent 协作：subagent 调度、Agent2Agent 通信、异步任务和结果回注。
  • Agent memory：长期记忆、知识图谱、向量检索、多跳推理和记忆代谢。
  • Agent x Hardware：桌宠、实体设备、本地自动化和多模态交互。
  • 工程化实践：模型降级、限流重试、上下文压缩、日志记录和回归测试。
